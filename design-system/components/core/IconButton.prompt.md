Circular chrome icon control for toolbars, composer send/attach, and sheet headers.

```jsx
<IconButton variant="chrome" size="lg">↑</IconButton>
<IconButton variant="soft" active>🔌</IconButton>
<IconButton variant="bare">✕</IconButton>
```

Variants: `soft` (default veil + hairline), `chrome` (filled plate — the composer send button), `bare` (no chrome). Pass `active` for toggled toolbar items. Built-in press-shrink.
