The machined-metal panel — obsidian fill, hairline edge, top specular highlight.

```jsx
<Card variant="raised">记忆条目</Card>
<Card variant="chrome">成本级联 · L0–L6</Card>
<Card variant="inset">日志 / 终端输出</Card>
```

Variants: `raised` (default card), `chrome` (brand glow — feature panels), `inset` (recessed well for logs/terminals), `flat` (subtle veil). Pass `onClick` to make tappable.
