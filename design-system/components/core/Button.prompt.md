Black God action button — a brushed-chrome plate on obsidian for the single most important action on a surface.

```jsx
<Button variant="primary" size="lg" block>允许执行</Button>
<Button variant="secondary" icon="🔌">接口设置</Button>
<Button variant="ghost">拒绝</Button>
<Button variant="danger">清除全部记忆</Button>
```

Variants: `primary` (chrome plate, default — one per surface), `secondary` (chrome-outline), `ghost` (bare hairline), `danger` (red plate). Sizes `sm | md | lg`. Use `block` for full-width sheet actions. Has a built-in press-shrink; don't wrap in another animated container.
