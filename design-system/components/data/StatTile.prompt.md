Compact dashboard metric tile — chrome gradient value, uppercase muted label, optional semantic delta.

```jsx
<StatTile label="本月省下" value="¥248" delta="↓ 62% vs 全用大模型" deltaTone="ok" />
<StatTile label="缓存命中" value="71" unit="%" icon="⚡" />
```
The value renders in the chrome gradient automatically. Keep labels short.
