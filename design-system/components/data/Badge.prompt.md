Functional state token. Semantic color (green/blue/amber/red) is reserved for status — never decoration.

```jsx
<Badge tone="ok" dot>已完成</Badge>
<Badge tone="run" dot>执行中</Badge>
<Badge tone="warn">需确认</Badge>
<Badge tone="chrome">L0 缓存命中</Badge>
```
Tones: `ok | run | warn | err | chrome | neutral`. `dot` adds a glowing status dot.
