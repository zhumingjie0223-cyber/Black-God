## 一句话

> **你要的不是别人的 AI，不是一次性聊天，而是一个真正属于你的、主动型的工作生活助理。**

它首次安装就在**本地认识你**、默认不上传云端；内部靠记忆、缓存、工具和分级模型调用**节约 Key**；内置**强沙箱**真实执行；可配置远端承担重任务；手机 **App 商城**把真实 App 接进端到端任务链；**技能系统**从真实任务里自己长出能力——**越用越懂你，越用越强。**

终点不是一个网页或 App，而是：**一个属于你的个人 Agent 操作系统（Agent OS）。**

## 一、核心是什么

Black God 的核心**不是炫工具，不是聊天机器人，也不是模型有多强**，而是一个问题：

> **它能不能通过本地记忆和行为画像，越来越懂你的工作和生活。**

它要懂的是你这个人：你怎么说话、工作里常做什么、有哪些项目/服务器/仓库、哪些能碰哪些不能碰、你喜欢快和直接、希望它主动接住上下文而不是每次重新解释。

### 三个核心特征

**1. 不是问答，是主动型助理**
```
你给一个目标
  → 它自己理解背景 → 先查记忆和缓存 → 能不用模型就不用
  → 需要模型才调用 → 能本地做就本地做，需要远端再上远端
  → 给你结果，而不是把问题丢回来
```

**2. 越用越省（不烧 Key）**
智能 = 本地记忆 + 缓存 + 工具 + 沙箱 + 远端 + 模型网关，**不是"一个更强的模型"**。
衡量硬指标：**单位任务的模型调用成本随使用时间下降。**

**3. 越用越强（自进化）**
每次任务完成 → 沉淀为记忆/技能/模板 → 下次更省更准。技能从真实任务里长出来，拒绝垃圾堆积。

## 四、六大核心子系统

### 1. 成本引擎 —— "不烧 Key"的级联拦截（✅ 已实测）
请求自上而下穿过 L0~L6，越下成本越高，尽量上层解决：
```
L0 规则/确定性  → L1 精确缓存 → L2 语义缓存 → L3 记忆检索
→ L4 技能命中  → L5 复杂度分类 → L6 模型网关
```
**实测结果**：11条请求，本地拦截率 **81.8%**，省Key **98.8%**（39.60 → 0.48）。

### 2. 模型网关 —— 你不该关心后面是谁（✅ 已实测）
带评分的路由决策，不是随机切：
```python
score(m) = w_cap·能力 + w_succ·成功率 - w_cost·成本 - w_lat·延迟 - w_ref·拒绝率
```
+ fallback 链（best→次选→降档→本地兜底）+ 熔断器（连续失败下线，半开探测恢复）+ 重试。

### 3. 记忆系统 —— 把经历变经验
压缩管线：原始 → 提取事实 → 合并去重 → 摘要 → 标重要度 → 写入 → 低价值遗忘。
四层：即时 / 短期 / 长期（含**禁碰清单**，pinned 永不遗忘）/ 技能。
遗忘评分：`retention = importance·confidence·exp(-Δt/τ)·log(1+hits)`
检索：向量 + 关键词混合（账号/路径/仓库名等实体走精确命中）。

### 4. 技能系统 —— 从真实任务里自己长出来
来源三种：任务沉淀 / 失败复盘 / 高频需求。**用过、验证过、可复用才配成为技能。**
质量闸门：低频不沉淀 · 一次性不沉淀 · 不稳定不沉淀 · 重复合并 · 过期淘汰 · 高价值强化。
生命周期：候选 → 启用 → 强化 ／ 淘汰 ／ 合并。

### 5. Agent Loop —— 给结果而不是丢回问题
带自我纠错的状态机：`PLAN → ACT → OBSERVE → REFLECT/CRITIC`
护栏：死循环守卫（max_iter + 状态指纹去重）· HITL 确认（高危操作走 /api/confirm）· checkpoint 可回滚 · 交付前验证步。

### 6. 人格层 —— 让 Agent 有稳定风格
不是角色扮演，是交互策略：**懂你 → 预判 → 少问 → 直接做。**
公开版用中性助手；**私人版保留"赵思涵"人格 + 完整偏好**（情绪识别/场景切换/主动延续话题）。

## 五、安全边界（决定能不能公开）

- **隐私**：本地读取·本地保存·本地分析·不上传云端。可以懂你，但不靠偷传数据懂你。
- **权限**：授权多少懂多少；未授权 → 陌生人模式，不编造画像。
- **执行**：默认沙箱隔离；每次工具调用/命令/文件改动留可追溯审计日志；高危操作二次确认；**禁碰清单是硬约束**。
- **红线**：能力只服务于**你自己的或已授权的系统**。不做未授权入侵、不碰他人系统。

> "Black" 代表深层执行力，不是无法无天。边界写得越清楚，越敢自用、越敢公开。
