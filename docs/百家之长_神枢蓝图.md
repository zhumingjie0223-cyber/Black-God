## 二、两根柱子：把百家熔成一枢

### 柱一 · 模型网关（执行层）— ONE ENTRY, ROUTE ALL BRAINS
上层永远只叫「神枢」，底层自动路由 **Claude / GPT / Gemini / Grok / Qwen / GLM / MiniMax / 本地**。
一家拒绝、限流、失效 → 自动换脑，绝不掉线。**百家的脑，一个入口尽收。**

### 柱二 · 枢语（统一层 · 独创）— THE META-LANGUAGE, NOBODY ELSE HAS IT
不同模型、不同模态，输出千差万别。**枢语把它们全部坍缩到同一套五维语义坐标**——
这是别家都没有、神枢独造的元语言。百家是零件，**枢语是让零件成为「一个神枢」的胶水与灵魂。**

**五维坐标**：核（语义内核）· 映（色彩投射）· 态（张力状态）· 标（指向）· 相（显隐）
