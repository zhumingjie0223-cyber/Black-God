## 三、神枢 Builder (造应用引擎)

### 3.1 功能
用户通过自然语言对话，神枢自动生成完整应用（网页/SaaS/工具），一键部署上线。

### 3.2 技术栈

| 层 | 技术 |
|----|------|
| 前端 | React + Remix SSR |
| 样式 | Tailwind CSS + shadcn/ui |
| AI 引擎 | 神枢 AI 网关 (Claude 主力) |
| 运行时 | 浏览器内 WebContainer (Node.js) |
| 编辑器 | CodeMirror 6 |
| 终端 | xterm.js |
| 部署 | Cloudflare Workers/Pages |
| 状态 | nanostores |

### 3.3 AI 工作流

```
用户: "帮我做一个XXX"
    │
    ▼
神枢 Builder 引擎:
  1. 解析意图 → 确定复杂度 (低/中/高/极)
  2. 路由到对应模型档位
  3. 生成结构化代码产物
  4. 安装依赖 → 启动开发服务器
  5. 实时预览
  6. 用户"满意"→ 一键发布
```

### 3.4 模型分档路由

| 档位 | 模型 | 场景 |
|------|------|------|
| 极低 | Haiku | 小改动/改文字/格式化 |
| 低 | Sonnet 低档 | 加个小功能 |
| 中 | Sonnet 中档 | 常规页面生成 |
| 高 | Sonnet 高档 | 复杂页面/SaaS功能 |
| 极高 | Opus 高档 | 架构设计/重构 |
| 兜底 | DeepSeek V4 | 批量/低成本任务 |
| 备份 | Gemini 2.5 | 主力厂挂了降级 |

## 五、神枢 Agent (任务执行引擎)

### 5.1 核心机制: todo.md 驱动

```
用户给任务
    │
    ▼
Planner 拆解:
  写 todo.md 清单
  ├── [ ] 步骤1
  ├── [ ] 步骤2
  └── [ ] 步骤3
    │
    ▼
Executor 循环执行:
  取第一个 [ ] → 执行 → 标记 [x]
  上下文太长时 → 重读 todo.md 找回目标
    │
    ▼
实时推送每步进度 → 用户可见
    │
    ▼
交付物 + 可部署上线
```

### 5.2 沙箱执行环境

| 环境 | 用途 |
|------|------|
| WebContainer | 浏览器内 Node.js |
| GCP 新加坡 | 真实 Linux 虚拟机，跑任意命令 |
| GCP 香港 | 渗透/逆向专用沙箱 |

## 六、神枢 AI 网关

### 6.1 架构

```
神枢 App
    │
    ▼
https://claude.lufei.uk (新加坡中转)
    │
    ├── /v1/messages        ← Anthropic 原生协议
    └── /v1/chat/completions ← OpenAI 兼容协议
    │
    ├── Claude API (主力)
    ├── OpenAI API (备胎)
    ├── DeepSeek API (降本)
    └── Google Gemini API (跨厂备份)
```

### 6.2 已接入模型

| 厂商 | 模型 | 上下文 | 用途 |
|------|------|--------|------|
| Anthropic | Opus 4.8 | 1M | 架构/复杂推理 |
| Anthropic | Sonnet 5 | 1M | 主力代码生成 |
| Anthropic | Fable 5 | 1M | 创意/写作 |
| OpenAI | GPT-5.6 | 128K | 通用对话 |
| OpenAI | GPT-5.4 | 128K | 常规任务 |
| DeepSeek | V4 Pro | 1M | 降本批量 |
| DeepSeek | V4 Flash | 128K | 快速响应 |
| Google | Gemini 2.5 Pro | 1M | 跨厂备份 |
| Google | Gemini 2.5 Flash | 1M | 快速备份 |

### 6.3 分档路由逻辑

```
任务分析 → 复杂度评分 (0-100)
    │
    0-20  → Haiku 或 DeepSeek Flash (最便宜)
    20-40 → Sonnet 低档 (便宜够用)
    40-60 → Sonnet 中档 (常规)
    60-80 → Sonnet 高档 (复杂)
    80-100 → Opus 高档 (最贵最强)
    │
    主力厂挂了 → Gemini 备份
    预算吃紧 → DeepSeek V4 兜底
```

## 九、枢语 (Shuyu-lang)

### 9.1 定位
Black God 底层的定制语言引擎，29.5亿语义空间。

### 9.2 技术栈

| 组件 | 文件 | 功能 |
|------|------|------|
| 词库 | lexicon.js | 词汇定义 |
| 语义引擎 | nexuslang.js | 语义计算 |
| 生成器 | gen.mjs | 内容生成 |
| Python引擎 | shuyu_engine.py | Python绑定 |
| 工作线程 | worker.mjs | 并行处理 |

### 9.3 与神枢的关系

```
用户自然语言
    │
    ▼
神枢 (AI Agent 操作系统)
    │  理解意图 → 拆解任务 → 调度工具
    │
    ▼
枢语 (语义空间引擎)
    │  精准语义匹配 → 代码生成 → 格式转换
```
