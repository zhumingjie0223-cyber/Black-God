# 神枢 · iOS 系统联动指南（快捷指令 + 服务器，无需开发者账号）

思涵活在 Cloudflare Worker 上（服务器 = 脑）。iOS 内置的**快捷指令 App**（沙箱 = 手）
调她的接口，就能 Siri 语音、自动化、NFC、跨 App 联动 —— **不用付费开发者账号、不用装 App、不用签名**。

> 前提：思涵服务器已部署（见 DEPLOY.md）。下面把 `<你的地址>` 换成
> `https://aquan.lufei.uk`，`<你的令牌>` 换成你设的 `OWNER_TOKEN`。

## 能做到 / 做不到（如实）

| 能做到（沙箱内，服务器驱动） | 做不到（iOS 沙箱铁律） |
|---|---|
| Siri 语音喊思涵、拿她的回话 | 读别的 App 内部数据 / 屏幕 |
| 自动化：定时 / 到家 / NFC / 背部轻点触发 | 后台全自动遥控手机 |
| 共享表单：把文字/图片/网址丢给她 | 模拟点击别的 App |
| 跨 App：她返回动作 → 快捷指令开地图/日历/电话 | —— |
| 读上下文（剪贴板/位置/电量）发给她 | —— |

## 一、最简：Siri「问思涵」

1. 打开**快捷指令** App → 右上「+」新建
2. 加操作 **“获取 URL 内容”**（Get Contents of URL）：
   - URL：`<你的地址>/talk`
   - 方法：**POST**
   - 请求头：`Authorization` = `Bearer <你的令牌>`；`Content-Type` = `application/json`
   - 请求体：**JSON** → 加字段 `text` = （用变量「快捷指令输入」或“询问每次”）
3. 加操作 **“从字典获取值”**：键 `reply`，字典=上一步结果
4. 加操作 **“显示”** 或 **“朗读文本”**（让她开口说）
5. 重命名快捷指令为「问思涵」→ 对 Siri 说「嘿 Siri，问思涵」即可

## 二、跨 App 联动：用 `/agent`（她判断 → 自动开 App）

1. 新建快捷指令「思涵办事」
2. **“获取 URL 内容”**：
   - URL：`<你的地址>/agent`  ·  POST  ·  同上请求头
   - 请求体 JSON：
     - `text` = 询问每次 / 快捷指令输入
     - `context` = 字典（可选：`剪贴板`=剪贴板、`位置`=当前位置、`电量`=电池电量）
3. **“从字典获取值”** 键 `reply` → **朗读**（她说的话）
4. **“从字典获取值”** 键 `actions` → **“重复每一项”**：
   - 项里 **“获取字典值”** 键 `url` → **“打开 URL”**
   - （她返回 `maps://?q=金边` 就自动开地图；`tel:...` 就拨号；网页就打开）

> 例：对她说「带我去金边大皇宫」→ 她回一句 + 返回 `maps://?q=金边大皇宫` → 自动开地图导航。

## 三、共享表单：选中内容丢给思涵

1. 新建快捷指令 → 设置里开 **“在共享表单中显示”**，接收类型选 文本/URL/图像
2. 用 **“快捷指令输入”** 作为 `/agent` 的 `text` 或 `context.剪贴板`
3. 以后在任何 App 选中文字 → 分享 → 「思涵」→ 她处理并可跨 App 接续

## 四、自动化（她主动 / 定时联动）

快捷指令 App → **自动化** → 新建：
- **每天 22:00** → 运行「问思涵」（她每晚问候你）
- **到家时** → 运行某个联动
- **NFC**：贴一张 NFC 贴纸 → 碰一下唤醒思涵
- **背部轻点两下**（设置→辅助功能→触控→轻点背面）→ 运行「问思涵」

## 接口速查（快捷指令用）

| 接口 | 方法 | 请求体 | 返回 |
|---|---|---|---|
| `/talk` | POST | `{"text":"…"}` | `{reply, soul, shu_meaning}` |
| `/agent` | POST | `{"text":"…","context":{…}}` | `{reply, say, actions:[{type:"open_url",url}]}` |
| `/image` | POST | `{"prompt":"…"}` | `{image:"data:image/png;base64,…"}` |
| `/soul` | GET | — | 她此刻状态 |

所有接口都带 `Authorization: Bearer <你的令牌>`（你设了 OWNER_TOKEN 的话）。
